災害時の患者・利用者の安否確認を3秒で実現する無料ツール

開発者：daichance.med
バージョン：v2.3.0

■ 【推奨】一番簡単な使い方
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

スマホでQRコードを読み取るだけで使えます。

または、このURLにアクセス：
https://daichance-med.github.io/zaitaku-bcp-hazardmap/

✅ ダウンロード不要
✅ ファイル保存不要
✅ iPhone/Android両対応
✅ 常に最新版が使える


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 【従来の方法】ダウンロードして使う
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

オフラインで使いたい場合や、従来の方法を好む場合：

1. index.html をダブルクリック
   → ブラウザで開く

2. 同意画面でチェックを入れて「同意して開始」

3. 患者データ（Excel）をアップロード

4. 地図が表示される


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 患者データの準備
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

patient_template_v2.2.2.xlsx を使用してください。

【必須項目】
- 患者名
- 住所（番地まで）
- 電話番号1、電話番号2
- リスク分類（高・中・低）
- 備考

【注意】
- 住所は番地まで正確に入力
- 電話番号は2つまで入力可能
  （3つ目以降は備考欄に記載）


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 使い方動画（4分）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

詳しい使い方は動画で確認できます：
https://youtu.be/ndd0DmPKOlI?si=kY48OtXxedGVmJC5


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 主な機能
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📍 患者住所の自動地図化
   国土地理院APIで高精度な位置標定

⚠️ ハザード情報の統合表示
   洪水・土砂災害・津波をワンクリック切り替え

🔴 リスク分類による色分け
   高（赤）・中（黄）・低（緑）で優先順位が一目瞭然

📞 ワンクリック電話発信
   スマホでタップするだけで即座に電話

📝 備考欄表示
   リスク分類の根拠や特記事項を記録

🔒 完全オフライン処理
   データを外部サーバーに送信しません

🔍 患者名検索機能
   地図表示中に患者氏名で即座に検索・ジャンプ

🎯 リスク別フィルター表示
   高・中・低リスクで絞り込み表示

📋 折りたたみ式UIパネル
   パネルを収納して地図を広く使用可能


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ データの保存
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

「地図データ保存」ボタンでJSONファイルとして保存できます。

次回以降は、保存したJSONファイルを読み込むだけで
すぐに地図が表示されます。

【重要】
JSONファイルには患者情報が含まれます。
取り扱いには十分ご注意ください。

- 全角・半角に注意

【iPhoneで開けない】
- URLまたはQRコードからアクセス（推奨）
- ファイルをメールで自分に送り、添付から開く
