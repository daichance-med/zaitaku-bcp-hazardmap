## ✨ 主な機能

- **📍 患者住所の自動地図化** - 国土地理院APIで高精度な位置標定
- **⚠️ ハザード情報の統合表示** - 洪水・土砂災害・津波をワンクリック切り替え
- **🔴 リスク分類による色分け** - 高（🔴）・中（🟡）・低（🟢）で優先順位が一目瞭然
- **📞 ワンクリック電話発信** - スマホでタップするだけで即座に電話（電話番号2つまで、3つ目以降は備考欄にメモ）
- **📝 備考欄表示** - リスク分類の根拠や特記事項を記録
- **🔒 完全オフライン処理** - データを外部サーバーに送信しません

---

## 📱 使い方（30秒で完了）

1. 上記URLまたはQRコードでアプリを開く
2. 患者データ（Excel）をアップロード
3. 地図が自動表示される
4. ハザードマップを重ね合わせて確認
5. JSON保存で次回以降は即座に表示

**詳しい使い方は2分の動画で確認：**
📹 [YouTube使い方動画](https://youtu.be/ndd0DmPKOlI?si=kY48OtXxedGVmJC5)

## 📂 患者データの準備

患者データは専用のExcelテンプレートで作成します：

[患者リストテンプレート_v2.2.2.xlsx](患者リストテンプレート_v2.2.2.xlsx) をダウンロード

### 必須項目：
- 患者名
- 住所（番地まで記入）
- 電話番号1、電話番号2
- リスク分類（高・中・低）
- 備考
