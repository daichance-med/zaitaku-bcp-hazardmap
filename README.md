# 在宅BCPハザードマップアプリ

**災害時の患者・利用者の安否確認を3秒で実現する無料ツール**

[![License: Custom](https://img.shields.io/badge/License-Custom-blue.svg)](LICENSE.txt)
[![Version](https://img.shields.io/badge/version-2.2.7-green.svg)](https://github.com/daichance-med/zaitaku-bcp-hazardmap/releases)

患者住所を地図上に自動表示し、洪水・土砂災害・津波などのハザード情報と統合して可視化します。
訪問診療・訪問看護・訪問介護はもちろん、入院・外来・通所等の利用者マップとしてもご活用いただけます。

---

## ✨ 主な機能

- 🗺️ **患者住所の自動地図化** - 国土地理院APIで高精度な座標変換
- ⚠️ **ハザード情報の統合表示** - 洪水・土砂災害・津波をワンクリック切り替え
- 📞 **ワンクリック電話発信** - スマホでタップするだけで即発信（電話番号2つまで。3つ目以降は備考欄にメモ）
- 🎨 **リスク分類による色分け** - 高（🔴）・中（🟡）・低（🟢）で優先順位が一目瞭然
- 📝 **備考欄表示** - リスク分類の根拠や特記事項を記録
- 💾 **JSON保存・読込** - 次回から3秒で地図を再表示
- 🔒 **完全オフライン処理** - データを外部サーバーに送信しません

### 2. 使い方（3ステップ）

**ステップ1：患者リストExcelを準備**
- テンプレートに患者情報を入力
- 電子カルテからCSVエクスポートも可能

**ステップ2：アプリにアップロード**
- `在宅BCPハザードマップアプリ.html` をブラウザで開く
- Excelファイルをドラッグ&ドロップ

**ステップ3：地図が自動表示！**
- 患者位置とハザード情報が一目でわかる
- マーカークリックで電話発信

詳細は [README.txt](README.txt) をご覧ください。

## 🚨 災害時の使用について

**✅ 通常の災害では問題なく使用可能**

台風・水害・地震（震度5-6程度）など、ほとんどの災害ではインターネット接続は維持されます。  
本アプリは災害時の安否確認に有効です。

**💡 このアプリが活躍する場面：**
- 道路冠水で訪問できない → 地図で患者位置を確認し電話で安否確認
- 地震発生直後 → ハザードマップで危険地域の患者を優先的に連絡
- 台風接近中 → 浸水予想区域の患者に事前連絡

**⚠️ 念のための準備：**

大規模災害で通信が途絶えた場合に備え、以下の準備を推奨します：

1. **地図のスクリーンショット保存**（スマホ・PCに保存）
2. **印刷版も用意**（A3推奨、ラミネート加工）
3. **JSON保存でバックアップ**（次回起動時に即座に復元可能）

→ これで通信障害時も対応可能です
